## 1. Introduction

The Gym Membership Management System is a desktop application developed using Java, Swing, and SQL. It is designed to help gyms efficiently manage their operations through a centralized system.

The application focuses on simplifying administrative tasks such as managing members, membership plans, and payment records. It allows administrators to track payments and maintain organized records using basic CRUD operations.

---

## 2. Functionalities

### User Authentication (Login)

The system allows administrators to log in using a username and password stored in the database. This ensures that only authorized users can access and manage the system.

### Member Management

Administrators can manage member details by adding, updating, deleting, and viewing records. Member information includes name, phone number, and assigned membership plans.

### Membership Plan Management

The system supports creating and managing different membership plans such as monthly, quarterly, and yearly plans. Each plan includes details like duration and price.

### Payment Management

The system records payments made by members. It stores information such as payment amount, payment date, and the associated member. This helps track financial transactions within the gym.

### Dashboard Navigation

A dashboard is provided to allow easy navigation between different sections of the system. It improves usability and efficiency when performing tasks.

### Report Generation

The system generates basic reports, including registered members and payment history, to provide insights into gym operations.
